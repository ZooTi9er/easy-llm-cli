- OPENAI_API_KEY 如何传递给 OpenAI SDK
- LLM_PROVIDER 如何影响认证类型判断

### 2. API 调用流程
- 用户输入 → Config → ContentGenerator → CustomLLMContentGenerator → OpenAI SDK → 自定义 API
- 响应返回路径和数据处理
- 错误处理和重试机制

### 3. 数据转换
- Gemini 格式 ↔ OpenAI 格式的双向转换
- 工具调用的数据格式处理
- 流式响应的数据流处理

### 4. 配置管理
- 环境变量的优先级和覆盖机制
- 动态配置的更新方式
- 配置验证和错误处理
